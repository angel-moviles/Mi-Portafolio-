FarmaPOS: Sistema de Gestión de Farmacia y Punto de Venta

El desafío: Las farmacias tradicionales gestionan su inventario de forma manual, lo que provoca pérdidas por productos caducados, errores en ventas y falta de trazabilidad. Además, los procesos de venta son lentos y no ofrecen herramientas analíticas para la toma de decisiones.

Mi solución: Diseñé e implementé FarmaPOS, un sistema completo de gestión que integra un backend en Laravel (API REST) y una aplicación móvil en React Native con Expo. El sistema permite control de inventario en tiempo real, ventas rápidas con escáner de código de barras, generación de tickets PDF y reportes analíticos.

Enfoque técnico

  • Arquitectura cliente-servidor – Separación clara entre frontend (React Native) y backend (Laravel API) para escalabilidad y mantenibilidad.
  • Base de datos relacional robusta – MySQL con 11 tablas interconectadas, índices, vistas y procedimientos almacenados.
  • Autenticación segura – JWT + Laravel Sanctum con control de acceso basado en roles (Administrador, Farmacéutico, Cajero, Auxiliar).
  • Notificaciones push – Alertas de stock bajo, recordatorios diarios y confirmación de ventas con vibración mediante Expo Notifications.
  • Reportes y gráficas – Generación de tickets PDF con Expo Print y gráficas interactivas con React Native Chart Kit.

Resultados obtenidos

FarmaPOS logró digitalizar por completo la operación de la farmacia, reduciendo errores en un 95%, acelerando las ventas a menos de 30 segundos por transacción y proporcionando visibilidad total del inventario con alertas automáticas. Los reportes generados facilitan la toma de decisiones gerenciales y el cumplimiento normativo.

Este proyecto representa mi capacidad para desarrollar soluciones completas, desde el análisis de requisitos hasta el despliegue, integrando buenas prácticas de seguridad, usabilidad y rendimiento.
